API OverviewLast updated: 2020-03-04

The MedCab predictive API allows access to a machine learning process for predicting strains of therapeutic herbal treatments. With limited endpoints, the API has a narrow range of options but is nonetheless powerful. Although authentication is required to register for and use the main service, the predictive API, written by the MedCab development team's Data Engineers is currently open.
